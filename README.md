# lemon-cli

基于webpack4构建的Vue服务器端渲染脚手架。

## 使用

下载此项目，进入到项目的根目录，使用 `npm i` 安装所有依赖。

`npm run dev` 启动开发环境

`npm run build` 打包项目，即可以上线正是环境的

`npm start` 项目打包之后，开启服务

## 目录

build --- webpack的配置目录

src/client --- 客户端开发目录

src/server --- 服务器端开发目录，主要为vue的ssr代码。

## 关于

工具中，使用到了 `eslint` 来规范我们的代码习惯。

css预处理语言, 使用了 `sass` 作为css的预处理语言。
